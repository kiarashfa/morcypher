# Morse Code Device

**Morcypher** (from *Morse* + *cypher*) is a virtual telegraph key for sending, decoding, and learning Morse code.

Tap, hold, type, or click to compose a message, then play it back as authentic CW audio or share it through a link. Convert freely between text and Morse in either direction, watch the live binary-tree decoder light up character by character, and sharpen your skills with a built-in drill mode (blind or visible).
