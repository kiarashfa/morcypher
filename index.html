Manual

A virtual telegraph key. Four input modes, one tree.

1 KEY — tap, hold, click, type

  • Spacetap = dot · hold = dash
  • L-click / R-clickon the pad = dot / dash
  • Tap / Holdon the pad = dot / dash (mobile)
  • A–Z 0–9live-type any letter and watch the path light up
  • · — ␣ ⌫buttons: insert dot, dash, word break, backspace

A short countdown bar fills under SIGNAL while you wait; when it completes, the buffer is decoded into a letter.

2 TEXT → MORSE — transmit a sentence

  • Typeany message, then hit Transmit to play it through the tree + audio
  • Speedopen Customize to adjust WPM (5 = beginner, 40 = expert)
  • Sharecopies a shareable link with the Morse encoded in the URL

3 MORSE → TEXT — paste raw morse

  • · —insert dot or dash buttons
  • spacebetween dot/dash groups separates letters
  • /or 3 spaces separates words
  • Transmitplays the decoded text through the tree + audio

Accepts ·, •, —, –, ., - interchangeably.

4 DRILL — practice mode

  • Listena random letter plays through the tree
  • Tapwhich letter you heard from the 4 choices
  • ↻replays the current letter
  • Poolswitch between A–Z, +0–9, or +punctuation
  • ModeBlind hides the code (train your ear); Visible shows it (learn by reading) — in Customize

Three correct in a row unlocks a streak badge.
